Шаг 0. Вводный контроль

Задание: Раскрой скобки: $(a + b)(a + b) =$ ?

Подсказка: умножь каждый член первого многочлена на каждый член второго.

🔍 Разберёмся, в чём ошибка

При умножении многочлена на многочлен нужно каждый член первого умножить на каждый член второго:

$(a+b)(a+b) = a·a + a·b + b·a + b·b = a^2 + 2ab + b^2$
📝 Дополнительное задание:

Раскрой скобки: $(x + 1)(x + 1) =$ ?

Шаг 1. Формула квадрата суммы

$(a + b)^2 = a^2 + 2ab + b^2$
Квадрат суммы = квадрат первого + удвоенное произведение + квадрат второго.

Задание 1.1: Раскрой скобки: $(x + 3)^2 =$ ?

🔍 Разберёмся, в чём ошибка

Вспомни формулу: $(a+b)^2 = a^2 + 2ab + b^2$. Здесь $a = x$, $b = 3$.

Подставляем: $x^2 + 2·x·3 + 3^2 = x^2 + 6x + 9$

Типичная ошибка: $x^2 + 9$ (забыли удвоенное произведение!)
📝 Дополнительное задание:

Раскрой скобки: $(y + 2)^2 =$ ?

Шаг 2. Формула квадрата разности

$(a - b)^2 = a^2 - 2ab + b^2$
⚠️ Обрати внимание: перед удвоенным произведением стоит «минус»!

Задание 2.1: Раскрой скобки: $(x - 4)^2 =$ ?

🔍 Разберёмся, в чём ошибка

Здесь $a = x$, $b = 4$. Применяем формулу квадрата разности:

$x^2 - 2·x·4 + 4^2 = x^2 - 8x + 16$
Типичная ошибка: $x^2 - 16$ (это формула РАЗНОСТИ квадратов, а не квадрат разности!)
📝 Дополнительное задание:

Раскрой скобки: $(y - 5)^2 =$ ?

Шаг 3. Определи знак

💡 Правило: знак перед $2ab$ такой же, как между $a$ и $b$ в скобках.

Задание: В выражении $(c - 6)^2 = c^2 \ldots 12c + 36$ какой знак должен стоять вместо «…»?

🔍 Разберёмся, в чём ошибка

Посмотри на исходное выражение: $(c - 6)^2$. Между $c$ и $6$ стоит минус.

Значит, и в формуле перед удвоенным произведением будет минус.

$(c - 6)^2 = c^2 - 12c + 36$
📝 Дополнительное задание:

Какой знак стоит перед удвоенным произведением в $(d + 7)^2 = d^2 \ldots 14d + 49$?

Шаг 4. Коэффициенты

⚠️ Типичная ошибка: $(2x)^2 = 4x^2$, а не $2x^2$!
Пример: $(2x + 3)^2 = (2x)^2 + 2·(2x)·3 + 3^2 = 4x^2 + 12x + 9$

Задание: Раскрой скобки: $(2x + 5)^2 =$ ?

🔍 Разберёмся, в чём ошибка

Здесь $a = 2x$, $b = 5$. Важно: в квадрат возводится всё выражение $2x$, а не только $x$!

$(2x)^2 + 2·(2x)·5 + 5^2 = 4x^2 + 20x + 25$
Типичная ошибка: $2x^2 + 20x + 25$ (забыли возвести 2 в квадрат!)
📝 Дополнительное задание:

Раскрой скобки: $(3x + 1)^2 =$ ?

Шаг 5. Свёртывание (обратное применение)

Формулу можно применять «справа налево»: из многочлена сделать квадрат двучлена.
Пример: $x^2 + 10x + 25 = (x + 5)^2$

Задание: Представь в виде квадрата двучлена: $m^2 + 6m + 9 =$ ?

🔍 Разберёмся, в чём ошибка

Алгоритм свёртывания:

  1. Найди квадрат первого члена: $m^2$ → это $m$
  2. Найди квадрат последнего члена: $9$ → это $3$
  3. Проверь удвоенное произведение: $2·m·3 = 6m$ ✓
Ответ: $(m + 3)^2$
📝 Дополнительное задание:

Представь в виде квадрата двучлена: $a^2 + 8a + 16 =$ ?

Шаг 6. Быстрые вычисления

Формулы помогают быстро считать квадраты чисел, близких к круглым.
Пример: $102^2 = (100 + 2)^2 = 10000 + 400 + 4 = 10404$

Задание: Вычисли: $21^2 =$ ? (используй формулу)

🔍 Разберёмся, в чём ошибка

Представь 21 как сумму: $21 = 20 + 1$. Примени формулу:

$21^2 = (20 + 1)^2 = 400 + 40 + 1 = 441$
📝 Дополнительное задание:

Вычисли: $31^2 =$ ?

Шаг 7. Уравнения

Пример: $(x + 3)^2 - x^2 = 21$
$x^2 + 6x + 9 - x^2 = 21$
$6x = 12$, значит $x = 2$

Задание: Реши уравнение: $(x + 4)^2 - x^2 = 40$. Чему равен $x$?

🔍 Разберёмся, в чём ошибка

Раскрой скобки по формуле: $(x+4)^2 = x^2 + 8x + 16$

Подставь: $x^2 + 8x + 16 - x^2 = 40$

Упрости: $8x + 16 = 40$ → $8x = 24$ → $x = 3$

📝 Дополнительное задание:

Реши уравнение: $(x + 2)^2 - x^2 = 12$. Чему равен $x$?

Шаг 8. Упрощение выражений

Задание: Упростите: $(a + 3)^2 - (a - 3)^2 =$ ?

Подсказка: раскрой обе скобки и приведи подобные.

🔍 Разберёмся, в чём ошибка

Раскроем обе скобки:

$(a+3)^2 = a^2 + 6a + 9$

$(a-3)^2 = a^2 - 6a + 9$

Вычтем: $(a^2 + 6a + 9) - (a^2 - 6a + 9) = a^2 + 6a + 9 - a^2 + 6a - 9 = 12a$

Типичная ошибка: забыть раскрыть скобки со знаком минус перед второй скобкой!
📝 Дополнительное задание:

Упростите: $(b + 2)^2 - (b - 2)^2 =$ ?

Шаг 9. Найди ошибку

❌ Ученик написал: $(3 + x)^2 = 9 + x^2$

Задание: Какой член пропущен? Напиши правильный ответ полностью.

🔍 Разберёмся, в чём ошибка

Ученик забыл удвоенное произведение $2·3·x = 6x$.

Правильно: $(3 + x)^2 = 9 + 6x + x^2$
📝 Дополнительное задание:

Найди ошибку и напиши правильный ответ: $(5 + y)^2 = 25 + y^2$

Шаг 10. Две переменные

Задание: Раскрой скобки: $(5a + 2b)^2 =$ ?

🔍 Разберёмся, в чём ошибка

Здесь $a = 5a$, $b = 2b$. Применяем формулу:

$(5a)^2 + 2·(5a)·(2b) + (2b)^2 = 25a^2 + 20ab + 4b^2$
